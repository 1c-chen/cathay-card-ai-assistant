# cathay-card-ai-assistant

# 信用卡智能權益助手

## 專案簡介
本專案為國泰世華 CMA 儲備幹部計畫 AI 商業分析組提案，透過 GenAI 打造信用卡智能權益助手，整合客服問答、權益決策輔助與個人化行銷推薦三大功能，解決信用卡資訊繁雜、個人化不足與需求難以承接等痛點。

## 功能說明

**功能一：智能客服問答**
- 即時回答卡片條款與操作問題
- 理解模糊提問，超越傳統 FAQ

**功能二：智能權益顧問**
- 結合消費情境，規劃最佳回饋策略
- 自然語言查詢，引導完成兌換流程

**功能三：個人化內容推薦**
- 依消費習慣匹配最相關活動
- 生成客製文案，主動於 App 推送

## 技術架構
- **框架**：LangChain
- **LLM**：OpenAI API
- **RAG**：向量資料庫（Chroma）
- **介面**：Streamlit
